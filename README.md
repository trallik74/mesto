# Проект: Место

## О проекте
Mesto это одностраничный, адаптивный фото-блог.

## Технологии
Проект создан при помощи технологий:
* HTML5
* CSS
* JS (Es6 Modules)
* ООП
* Webpack
* API
* Файловая структура БЭМ Nested

Ссылка на проект [https://trallik74.github.io/mesto/](https://trallik74.github.io/mesto/)
